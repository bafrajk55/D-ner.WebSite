# Doner.WebSite
Modern responsive restaurant website for Nordgrill built with HTML, CSS and JavaScript. Features multilingual support (TR/DE/EN), interactive menu filtering, product modals, WhatsApp ordering, Google Maps integration and premium glassmorphism UI.

🔥 Nordgrill Restaurant Website

A modern and responsive restaurant website built using HTML, CSS and JavaScript.

Features

- 🌍 Multilingual (Turkish, German, English)
- 📱 Responsive Design
- 🍔 Interactive Menu Filter
- 🛒 Product Detail Modal
- 💬 WhatsApp Ordering
- ⭐ Google Reviews Slider
- 📈 Animated Statistics
- 📍 Google Maps Integration
- ✨ Glassmorphism UI
- ⚡ Smooth Animations

Technologies

- HTML5
- CSS3
- JavaScript (Vanilla)
